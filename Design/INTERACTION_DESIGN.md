# RepoGalaxy 交互设计文档

> 状态：设计中 | 最后更新：2026-03-13
> 核心概念：慵懒漂浮气泡云 + 拖拽聚类 + 分层导航

---

## 1. 视觉编码确认

### 1.1 最终映射方案

```
┌─────────────────┬─────────────────────┐
│ 视觉通道         │ 数据维度             │
├─────────────────┼─────────────────────┤
│ 闪烁频率         │ 综合活跃度指数        │
│ 呼吸缩放         │ Fork 数量            │
│ 圆形数量         │ 随机/无限滚动        │
└─────────────────┴─────────────────────┘

综合活跃度指数计算：
活跃度 = 0.4 * Commit频率 + 0.3 * Issue/PR活跃度 + 0.3 * Star增速

Fork 映射：
• 0 Fork:      不呼吸
• 1-100:       呼吸 1.0→1.05 (2秒周期)
• 100-1k:      呼吸 1.0→1.1 (3秒周期)
• >1k:         呼吸 1.0→1.15 + 光晕 (4秒周期)
```

## 2. 默认状态：慵懒漂浮 (Android 4.3 Jelly Bean 彩蛋风格)

### 2.1 漂浮动画规格

```
参考：Android 4.3 彩蛋里的糖豆 (BeanBag)

行为特征：
• 每个气泡有独立的速度向量 (vx, vy)
• 速度范围：-0.5 ~ 0.5 px/frame (非常缓慢)
• 边界反弹：碰到屏幕边缘柔和反弹
• 相互避让：轻微排斥力 (避免完全重叠)
• 随机扰动：偶尔改变方向 (模拟慵懒感)

物理参数：
├── 摩擦力：0.99 (几乎无摩擦，持续漂浮)
├── 弹性系数：0.8 (边界反弹柔和)
├── 质量：与 Star 数成正比 (大泡更惰性)
├── 避让半径：气泡半径之和 + 10px
└── 避让力度：与距离成反比 (轻微)

慵懒感营造：
• 极慢速度 (像在水中漂浮)
• 柔和曲线运动 (贝塞尔路径)
• 偶尔停顿 (模拟观察)
• 相互轻轻推开 (有生命的群体)
```

### 2.2 数据生成与缓存策略

```
多数据源设计：

数据源 A：GitHub Trending
├── 更新频率：每小时
├── 缓存时长：1小时
└── 优先级：高

数据源 B：GitHub Search (推荐算法)
├── 更新频率：每日
├── 缓存时长：24小时
└── 优先级：中

数据源 C：用户个性化推荐
├── 更新频率：实时计算
├── 缓存时长：15分钟
└── 优先级：最高

数据源 D：随机发现 (冷门宝藏)
├── 更新频率：每次启动
├── 缓存时长：会话级
└── 优先级：低

懒加载策略：
1. 启动时显示缓存数据 (立即响应)
2. 后台静默刷新数据源 A/B
3. 数据源 C 按需计算
4. 数据源 D 随机抽样

缓存配置：
┌────────────────────────────────────────────┐
│ 缓存设置                                    │
├────────────────────────────────────────────┤
│ 缓存目录：  [~/Library/Caches/RepoGalaxy] │
│ 最大大小：  [2] GB         [清理]          │
│ 缓存策略：  ● 智能清理   ○ 永久保留        │
│ 自动清理：  超过 80% 时自动清理            │
└────────────────────────────────────────────┘

手动清理：
• 清理按钮在设置中
• 确认对话框
• 清理时显示进度
```

## 3. 鼠标悬停：气泡停下

```
交互细节：

触发：鼠标进入气泡半径 + 10px
动作：
  1. 气泡逐渐减速 (200ms, EaseOut)
  2. 停止漂浮 (保持当前位置)
  3. 轻微放大 1.05x (100ms)
  4. 其他气泡轻微变暗 (80% 亮度)
  5. 显示 Tooltip (延迟 300ms)

视觉反馈：
• 气泡边缘出现微妙光晕
• 内部扇形颜色略微饱和
• Tooltip 淡入显示详情

恢复：鼠标离开
  • 气泡恢复漂浮 (随机新方向)
  • 缩小回 1.0x
  • 其他气泡恢复亮度
```

## 5. 拖拽 + 摇晃：聚类魔法

### 5.1 拖拽行为

```
拖拽开始：
• 左键按下并移动 5px 触发
• 气泡放大 1.1x
• 其他气泡变暗 60%
• 气泡跟随鼠标 (有轻微弹性延迟)

拖拽中：
• 气泡在拖拽下继续漂浮 (保持慵懒感)
• 显示轨迹尾迹 (可选)
• 经过的气泡轻微避让
```

### 5.2 摇晃检测

```
摇晃定义：
• 在 500ms 内，鼠标水平方向移动 3 次以上
• 或角速度超过阈值
• 触发聚类检索

视觉反馈：
• 被拖拽气泡开始发光 (亮度增加)
• 颜色向 Accent 色偏移
• 显示"摇晃中..."提示
```

### 5.3 聚类检索逻辑

```
触发条件：摇晃后松开左键

检索逻辑：
1. 获取被拖拽气泡的仓库信息
2. 提取关键特征：
   • 编程语言 (Primary Language)
   • Topics 标签
   • 描述关键词 (TF-IDF)
3. 搜索同类项目：
   • 相同语言 → 权重 0.4
   • 相同 Topics → 权重 0.3
   • 相似描述 → 权重 0.2
   • 相同 Owner → 权重 0.1
4. 按相似度排序
5. 选择最多 15 个最相似项目

动画效果：
• 被拖拽气泡继续放大 (1.0 → 1.5x, 300ms)
• 同类小气泡被"吸引"飞向大气泡 (500ms, EaseIn)
• 小气泡逐个融入大气泡 (stagger 50ms)
• 大气泡颜色变为渐变色 (代表聚类)
• 显示 "+N" 计数徽章
```

### 5.4 聚类后的大气包

```
大气泡外观：

┌───────────────────────────────────────────┐
│                                           │
│        ╭──────────────────╮              │
│       ╱  🔵🟢🔵⚪🟢🔵    ╲             │
│      │  🔵🟢⚪🟢🔵🟢⚪   │            │
│       ╲  🟢🔵⚪🔵🟢🔵    ╱             │
│        ╰──────────────────╯              │
│                                           │
│        中心仓库名                         │
│        [+12]  ← 包含12个子项目            │
│                                           │
│        脉动效果 (所有子项目呼吸同步)       │
│                                           │
└───────────────────────────────────────────┘

大气泡特性：
• 大小：根据包含项目数 (1.5x ~ 2.5x)
• 内部：子项目的扇形颜色混合
• 标签：显示 "+N" 计数
• 漂浮：质量更大，移动更慢
• 闪烁：活跃度综合计算
```

## 7. 长按破裂：解散聚类

### 7.1 长按检测

```
触发条件：
• 左键按住大气泡不动
• 持续 3 秒 (可设置)

视觉反馈 (倒计时)：

时间线：
0.0s ────── 正常状态
0.5s ────── 边框开始闪烁 (黄色)
1.0s ────── 边框变橙色
1.5s ────── 边框变红色
2.0s ────── 内部小气泡开始抖动
2.5s ────── 显示倒计时数字 "3, 2, 1"
3.0s ────── 破裂！

颜色动画：
• 0-1s: 正常 → 黄色
• 1-2s: 黄色 → 橙色
• 2-3s: 橙色 → 红色
• 3s+: 破裂效果

设置选项：
┌──────────────────────────────┐
│ 长按破裂时间                 │
│ [● 3秒] [○ 2秒] [○ 5秒]      │
└──────────────────────────────┘
```

### 7.2 破裂动画

```
破裂序列 (800ms)：

1. 膨胀阶段 (0-200ms)
   • 大气泡快速膨胀到 1.3x
   • 内部小气泡剧烈抖动
   • 颜色变为警告红

2. 破裂瞬间 (200-250ms)
   • 大气泡"炸裂"成碎片
   • 小气泡向四周弹射
   • 粒子效果 (可选)

3. 散开阶段 (250-600ms)
   • 小气泡以爆炸速度向外飞散
   • 逐渐减速 (摩擦力)
   • 恢复各自漂浮状态

4. 稳定阶段 (600-800ms)
   • 小气泡恢复正常漂浮
   • 速度衰减到默认慵懒状态
   • 背景亮度恢复

物理参数：
• 爆炸初速度：随机 5-10 px/frame
• 方向：360度均匀分布 + 随机扰动
• 摩擦力：0.95 (快速减速)
• 恢复时间：约 2 秒回到正常漂浮
```

## 8. 状态切换：Pill 接管

```
Pill 栏切换视图模式：

默认：气泡云视图 (Lazy)
┌──────────┬──────────┬──────────┐
│ 💭 气泡云 │ 📋 列表  │ 🗺️ 星图  │
└──────────┴──────────┴──────────┘

切换逻辑：

1. 切换到 [📋 列表]
   • 气泡停止漂浮
   • 对齐到网格 (动画过渡)
   • 变形为卡片列表
   • 支持排序和筛选

2. 切换到 [🗺️ 星图]
   • 气泡根据时间/Star重新排列
   • 显示连接线 (关系图)
   • 支持缩放和平移

3. 切换回 [💭 气泡云]
   • 恢复慵懒漂浮状态
   • 随机散开
   • 聚类状态保留 (如果有)
```

## 10. 性能与体验优化

### 10.1 动画性能

```
GPU 加速：
• 所有位置变化使用 transform
• 避免 layout thrashing
• 使用 will-change 提示

对象池：
• 气泡对象复用
• 避免频繁创建/销毁

节流策略：
• 拖拽时降低物理计算频率
• 远离视口的气泡暂停动画
```

### 10.2 体验细节

```
声音反馈 (可选)：
• 聚类成功：轻微"啵"声
• 破裂：轻微"啪"声
• 点击：轻微"滴"声

触觉反馈 (MacBook)：
• 聚类时轻触
• 破裂时重触

辅助功能：
• 减少动画模式
• 高对比度模式
• 键盘导航支持
```
