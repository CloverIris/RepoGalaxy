# RepoGalaxy roadmap

## Current foundation

- .NET 10 and Avalonia 12.1.0
- Official Fluent theme with no third-party UI kit
- Migrated v3 SQLite store with WAL, Unix-millisecond instants, integrity checks, backup and recovery
- Bounded L1 plus compressed SQLite stale-while-revalidate cache with settings and statistics
- Verified authentication session, DPAPI credential envelope, resumable pagination and single-concurrency request priority
- Persisted two-stage ranking, diversity, deterministic exploration and impression feedback
- Responsive boards, two-column discovery grid, local contribution heatmap and official GitHub news rail

## Next increments

1. Complete first-run interest onboarding and persist the user's topic, language, and keyword profile.
2. Surface collection, tag, and note editing in the Library UI.
3. Add Windows 11 notification activation-to-detail navigation and package lifecycle testing.
4. Add conditional-request fixtures for every GitHub and RSS endpoint and long-running fault-injection tests.
